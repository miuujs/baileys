##

- bail does not require Selenium or any other browser to interface with WhatsApp Web, it does so directly using a **WebSocket**.
- Not running Selenium or Chromium saves you like **half a gig** of ram :/
- bail supports interacting with the multi-device & web versions of WhatsApp.
- Based on [@whiskeysockets/baileys](https://github.com/WhiskeySockets/Baileys) v7.0.0-rc10 with extended interactive message types (Native Flow Buttons, List, Legacy Buttons, Template, Carousel, Collection) and Meta AI rich response handling.
- Thank you to [@pokearaujo](https://github.com/pokearaujo/multidevice), [@Sigalor](https://github.com/sigalor/whatsapp-web-reveng), and [@Rhymen](https://github.com/Rhymen/go-whatsapp/) for their research and implementations.
